GOTO Příkazy

; Vysvětlení příkazů pro skoky v programu:
GOTO   - Nepodmíněný skok na určené návěští nebo číslo řádku
        N10 GOTO N100     ; Skočí na řádek N100

GOTOF  - Podmíněný skok vpřed (hledá návěští vpřed)
        N10 GOTOF N55     ; Skočí vpřed v programu na řádek N55

GOTOB  - Podmíněný skok vzad (hledá návěští vzad)
        N50 GOTOB N10     ; Skočí vzad v programu na řádek N10

; Příklad použití GOTO příkazů - Smyčka opakování 3x
N100 G0 X0 Y0      ; Počáteční pozice
N110 R1=3          ; Počet opakování (nastavení parametru R1 na hodnotu 3)
N120 G01 X50 F100  ; Pohyb v ose X
N130 G01 Y20       ; Pohyb v ose Y
N140 R1=R1-1       ; R1=3-1 (nastavení parametru R1 na hodnotu 2)
N150 IF R1>0 GOTOB N120  ; Jestliže R1 je větší než 0, skok nazpět na N120
N160 M30           ; Konec programu

TRANS Příkazy

; TRANS - Programovatelné posunutí nulového bodu pro karusel
; Základní příkazy:
TRANS X100            ; Posunutí nulového bodu na pozici X=100
TRANS Z50             ; Posunutí nulového bodu na pozici Z=50
TRANS X75 Z25         ; Kombinované posunutí na X=75 a Z=25
TRANS X0 Z0           ; Vynulování posunutí (návrat na výchozí nulový bod)
ATRANS X10            ; Přídavné posunutí o +10mm v ose X
ATRANS Z20            ; Přídavné posunutí o +20mm v ose Z

; Příklad 1 - Soustružení více stejných průměrů:
N100 G0 X100 Z5      ; Výchozí pozice
N110 TRANS X0 Z0     ; Výchozí nulový bod
N120 G1 X50 F0.2     ; Soustružení průměru
N130 G0 X100         ; Odjezd
N140 TRANS Z30       ; Nastavení nového nulového bodu na Z=30
N150 G1 X50 F0.2     ; Soustružení druhého průměru
N160 G0 X100         ; Odjezd

; Příklad 2 - Opakované soustružení s přídavným posunutím:
N200 R1=0            ; Počítadlo
N210 TRANS Z0        ; Výchozí pozice v Z
N220 G1 X40 F0.15    ; Soustružení průměru
N230 G0 X80          ; Bezpečná vzdálenost
N240 ATRANS Z25      ; Přídavné posunutí o +25mm v Z
N250 R1=R1+1         ; Zvýšení počítadla
N260 IF R1<3 GOTOB N220  ; Opakování 3x
N270 TRANS X0 Z0     ; Reset na výchozí nulový bod
N280 M30             ; Konec programu
